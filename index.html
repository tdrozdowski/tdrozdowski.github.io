Welcome to xymox.net.

What is xymox.net? Its my private domain that I've heald since 1996 or so. I've not done too much with it, but with the advent of Github and social programming, I've brought it back to life here. Xymox is reference to the gothic-electro band Clan of Xymox from the Netherlands. They have nothing to do with this site, except provide inspiration through their great music.

What will you find here? This site is a collection of various GitHub projects, discussions and examples of various things that I find interesting in the software world. Not quite a blog, but a link to things I'm working on in GitHub that are kinda cool and interesting.

The underlying technologies of what you find here will change from time to time. Currently, I am an advocate of the Play Framework and the Typesafe Stack with Scala. I am also an advocate of AngularJS, CoffeeScript and Yeoman.

Desert Code Camp - Nov 2013

I have presented a session at the Desert Code Camp on Nov 9th, 2013 entitled: Play Framework with Scala in Action. This is my first attempt at giving such a presentation, so hopefully all goes well. :)

Requirements:

  • JDK 6 or JDK 7 (for JVM)
  • Scala 2.10.x (for REPL examples)
  • Play Framework 2.2.x
  • A text editor (VI, EMACs, Sublime) or IDE (IntelliJ 13 EAP with Scala & Play 2 plugins was used building these examples)
  • A MongoDB instance (MongoHQ has a nice, free sandbox instance with 512MB)

NOTE: Mac uses can use HomeBrew to install most of the components listed above:

$ brew update
$ brew install scala
$ brew install play

I have a series of Gists that I will be using in the course, they are listed here:

  • Scala Basics via the REPL
  • Play Framework JSON API & WS API
  • Building a RESTful API with Play 2.2.x, ReactiveMongo and Scala
  • Adding CORS support to Play 2.2.x
  • Adding Basic Auth support to Play 2.2.x
